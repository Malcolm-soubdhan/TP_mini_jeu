-Travail réalisé par Enzo Langet et Malcolm Soubdhan
( octobre 2021)

-DESCRIPTION: 
Mini-jeu de combat au tour par tour, élaboré en PHP. Il existe 2 classes de personnages au choix: 
magicien ou guerrier, qui possèdent des caractéristiques ( PV, force, attaques..) différentes. 

-CONTEXTE: 
TP dans le cadre de la 2ème année de formation 
en bachelor développeur web à l'école Hetic. 

-STRUCTURE DU CODE:
PHP orienté objet


-FONCTIONNALITES PRESENTES: 

-> Sélectionner un personnage dans la banque de personnages présents sur la page d'accueil. 
-> créer un autre personnage, parmi les 2 types possibles.
-> L'enregistrer dans une base de données. 
-> Sélectionner un opposant à attaquer. 


-FONCTIONNALITES A DEVELOPPER: 

->Coder la fonction d'attaque
->Coder la fonction d'endormissement
-> Mettre en place un système d'expérience
-> Elaborer un système de repos
-> Permettre la suppression de personnage ( depuis le menu principal)
